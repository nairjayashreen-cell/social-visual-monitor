# Social Visual Brand Monitoring System

AI-powered ORM monitoring dashboard for:
- Instagram
- Facebook
- LinkedIn

## Features

- Logo similarity detection
- OCR text extraction
- Fraud keyword detection
- Risk scoring
- CSV/XLSX export
- Evidence management

## Setup

### Backend

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

Open:
frontend/index.html
